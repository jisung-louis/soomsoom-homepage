## 🛠️ 기술 스택

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM
- **UI Components**: Radix UI
- **Icons**: Lucide React

## 🏃‍♂️ 로컬 개발

1. 의존성 설치:
   ```bash
   npm install
   ```

2. 개발 서버 시작:
   ```bash
   npm run dev
   ```

3. 브라우저에서 `http://localhost:3000` 접속

## 📦 빌드 및 배포

```bash
# 프로덕션 빌드
npm run build

# 빌드 미리보기
npm run preview

# GitHub Pages 배포 (자동)
git push origin main
```
